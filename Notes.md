# Python Basics Notes

## 1. Variables

A variable stores a value.

Example:

```python
name = "Anika"
age = 18
```

Rules:

- Use letters, numbers, and underscore.
- Do not start with a number.
- Do not use Python keywords like `class`, `if`, `for`.

## 2. Data Types

Common Python data types:

- int: Whole numbers like 10, -3
- float: Decimal numbers like 3.14
- str: Text like "Hello"
- bool: True or False
- list: Ordered, changeable collection like [1, 2, 3]
- tuple: Ordered, fixed collection like (1, 2, 3)
- set: Unordered unique values like {1, 2, 3}
- dict: Key-value pairs like {"name": "Anika"}
- NoneType: No value, written as None

## 3. Operators

Arithmetic operators:

- + add
- - subtract
- * multiply
- / divide
- // floor divide
- % remainder
- ** power

Comparison operators:

- == equal
- != not equal
- > greater than
- < less than
- >= greater than or equal
- <= less than or equal

Logical operators:

- and
- or
- not

## 4. Type Casting

Type casting means converting one data type to another.

Example:

```python
age_text = "25"
age_number = int(age_text)
```

Common casting functions:

- int()
- float()
- str()
- bool()

## 5. Input and Output

- Use print() to show output.
- Use input() to take input from user.

Example:

```python
name = input("Enter your name: ")
print("Hello", name)
```
