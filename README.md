# Python Understanding

This repository contains beginner-friendly Python practice files.

## Project Structure

- `RoadMap.md` - Learning roadmap.
- `Python_Basics/` - Core Python topic files and mini projects.

Inside `Python_Basics/`:

- `Notes.md`
- `Variables.py`
- `DataTypes.py`
- `Operators.py`
- `Input_Output.py`
- `TypeCasting.py`
- `Exercises/`
- `Projects/`
  - `Calculator.py`
  - `BMI_Calculator.py`
  - `Age_Calculator.py`
  - `Temperature_Converter.py`

## How to Run a Python File

From the project root, run:

```powershell
python Python_Basics\DataTypes.py
```

Replace the file name with any file you want to test.

## Goal

Practice Python fundamentals topic by topic, then apply them in small projects.
